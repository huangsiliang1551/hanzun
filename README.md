# hanzun

## Quick Start

- Install dependencies for scripts: `npm install`
- Build admin static bundle:
  - `npm run dev:admin`
  - `npm run build:admin`
- Backend check:
  - `npm run check:php`
- Full check:
  - `npm run check`

## Environment

- Backend template: `backend/.env.example`
- Admin template: `admin-v2/.env.example`
